### Сборка и запуск
1) > docker compose build
2) > docker compose up -d

Пробрасываются порты 5432 и 8080, так что они должны быть свободны (ну или можно перенастроить маппинг в Docker-compose.yml).

### Тестирование

В директории otushw-test-data лежит JMeter проект с запросами. Для работы требуется установить plugins-manager (jarник лежит там же на всякий случай, а потом через него плагин Random CSV Data Set).
